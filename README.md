# Sports Economic Impact Simulation

このリポジトリは、以下の論文における数値シミュレーションを行うためのPythonコードを含んでいます。

**論文タイトル:** スポーツチームの経済波及効果に関する数理モデル分析  
**著者:** 政村 悠登 (東京大学大学院数理科学研究科)

本コードでは、高山–赤松–石倉 [3] の空間応用一般均衡モデルに基づき、ある地域にスポーツチームが参入した際の、各地域・各産業の企業の利潤と地域厚生の変化を計算します。

## Requirements

このコードは Python 3 系で動作します。実行には以下のライブラリが必要です。

* numpy
* scipy
* matplotlib

## Files

* `simulation.py`: 数値計算を行うメインスクリプトです。コンソールに計算結果（利潤の変化など）を出力します。
* `simulation_graph.py`: 特定の条件下でのシミュレーションを行い、結果をグラフとして画像保存します。
* `supplementary_note.pdf`: モデルの導出や計算に関する数理的な補足説明資料です。
* `data/`: 論文中の事例（愛媛県松山市・今治市）の計算に使用したデータセットが含まれています。

## Usage

### シミュレーションの実行 (simulation.py)

ターミナル（またはコマンドプロンプト）で以下のコマンドを実行してください。

```bash
python simulation.py
```

**パラメータの変更方法:** 入力データやモデルのパラメータ（地域数 A、産業数 I、代替弾力性 sigma など）は、スクリプト内に直接記述されています（ハードコーディング）。 条件を変更してシミュレーションを行う場合は、`simulation.py` を開き、以下の変数を適宜書き換えて保存してから実行してください。

```python
# 例: simulation.py 内のパラメータ設定部分
I = 7       # 産業数
A = 2       # 地域数
sigma0 = 5  # 代替弾力性
...
```
